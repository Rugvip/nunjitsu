# Nunjitsu

Nunjitsu is a native template engine aiming for full compatibility with
[Nunjucks](https://mozilla.github.io/nunjucks/), with a more secure execution
model and faster runtime performance.

The project is in its earliest stage. Its primary goals are:

- Compatibility with existing Nunjucks templates and behavior.
- Secure execution of untrusted templates.
- Fast startup and rendering through a native implementation.
- A straightforward migration path for existing Nunjucks users.
